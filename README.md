- Client, server それぞれで動くコードを書く
    - TCP で通信する
    - 言語は C
    - Client, server 間で何かしらのやりとりを行う実装にする
    - 次のライブラリ関数を使う事を推奨:
        - socket
        - bind
        - listen
        - accept
        - read
        - write
        - etc.
    - 関数呼び出し時に発生したエラーに対処するような実装にする
    - 上記に加え、さらに余裕があれば:
        - thread を使って高速化
        - シグナルでサーバー停止
- OS は Unix 系 (Windows なら WSL 使う)
- github で提出する
    - 提出前にコードが期待通りに機能し、通信の疎通ができていることの確認は必須
- 分からないことがあったらこのチャンネルに質問する
